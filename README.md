# 🦊 Zorro Diabetic Rehab & Fitness Website

## 📋 Project Overview

Zorro Diabetic Rehab & Fitness website is a comprehensive platform developed to enhance user engagement and accessibility for individuals seeking diabetic rehabilitation and fitness services.

⏳ **Duration**: July 2024 - August 2024

## 🚀 Features

- 🏋️‍♀️ Detailed information about diabetic rehab programs
- 💪 Fitness routines tailored for diabetic patients
- 📅 Appointment scheduling system
- 📱 Responsive design for all devices
- 🎨 Engaging animations and transitions

## 🛠️ Technologies Used

- ⚛️ React
- 🏗️ Vite
- 🎨 Tailwind CSS
- 🎭 Framer Motion
- 🚀 Vercel (for deployment)

## 🏗️ Installation

1. Clone the repository:
   ```
   git clone https://github.com/Sandheep-S-95/ZORO_Landing_Page.git
   ```
2. Navigate to the project directory:
   ```
   cd ZORO_Landing_Page
   ```
3. Install dependencies:
   ```
   npm install
   ```

## 🏃‍♂️ Running the Project

To run the project locally in development mode:

```
npm run dev
```

This will start the Vite development server. Visit the URL shown in your terminal (usually `http://localhost:5173`) to view the website.

To build the project for production:

```
npm run build
```

To preview the production build:

```
npm run preview
```

## 🧪 Testing

If you have set up testing in your Vite project, you can run the test suite with:

```
npm test
```

Note: Vite doesn't include a test runner by default. If you haven't set up testing, you may need to install and configure a test runner like Vitest.

## 🚢 Deployment

The project is configured for easy deployment on Vercel. Simply connect your GitHub repository to Vercel for automatic deployments.
